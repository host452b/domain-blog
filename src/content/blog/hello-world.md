这是第一篇博文，用来测试中文排版和整体布局。

## 为什么写博客

写作是思考的延伸。把想法写下来，迫使你把模糊的直觉变成清晰的论述。

## 一段代码

```typescript
const greet = (name: string): string => {
  return `你好，${name}`;
};
```

这段代码没有什么特别的，只是测试 syntax highlighting 是否正常工作。

## 混合语言

有时候用中文写，有时候用 English，有时候 mix together。This is intentional — 语言跟着思维走，不需要切换。
